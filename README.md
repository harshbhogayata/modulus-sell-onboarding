# Modulus Sell - Web Onboarding Prototype

This is a concept prototype for the **Modulus Sell** store onboarding flow on the web. It is built to demonstrate a frictionless, app-like registration process for shop owners.

## Features

- **Progressive Disclosure**: Steps are broken down (Phone -> OTP -> Details -> Location) to reduce cognitive load.
- **Visual Categories**: Uses large icons for business categories instead of text dropdowns.
- **Glassmorphism UI**: Modern aesthetic with smooth transitions using Framer Motion.
- **Mobile-First Design**: Fully responsive layout that works on desktop and mobile browsers.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS v4**
- **Framer Motion**

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the link (usually http://localhost:5173 or 5174).
